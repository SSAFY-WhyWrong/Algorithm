## Baekjoon 3020 개똥벌레
### **Idea**
* 높이별로 석순과 종유석이 몇 개를 만나는지 세준다
* 종유석은 기존의 높이보다 +1 해서 기록해준다 (범위를 int로 했으니 0.5 구간이 없지만 임의로 있는 것 처럼 두기 위해 차이를 둔다)
* 1부터 최대 높이까지 중에서 종유석과 석순의 합이 제일 적은 구간을 찾는다.

### ** 핵심 부분 구현**
```java
	for(int h=1;h<=H;h++) {
		if(result > up[h] + down[h]){
			result = up[h] + down[h];
			cnt = 1;
		}else if(result == up[h]+down[h]) {
			cnt++;
		}
	}
```
