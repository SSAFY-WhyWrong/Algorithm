## 1. 이진수

- 원래 풀이

  ```
  for(int i=2;i<=N;i++) {
      zeroCnt = length - zeroCnt;
      length *= 2;
  }
  ```

- 2의 1000승까지 가기 때문에 오버플로우 발생 

**BigInteger**

- 문자열 형태로 저장되어서 어떤 숫자이든지 저장할 수 있다.

- 초기화 할 때에는 문자열을 인자 값으로 넘겨줘야 한다.



## 2. 개미굴

**TreeMap** 

- 객체를 저장하면 자동으로 정렬된다. (기본 오름차순 정렬)

- 레드-블랙 트리(?)로 이루어져 있다.

  - 부모 노드보다 작은 값을 가지는 노드는 왼쪽으로 큰 값은 오른쪽으로 배치
